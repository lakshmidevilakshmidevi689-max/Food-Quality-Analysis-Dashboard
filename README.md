# 🍽️ Food Quality Analysis Dashboard

## 📌 Overview

This project is an interactive dashboard developed using **Python**, **Streamlit**, **Pandas**, and **Plotly** to analyze food quality datasets and visualize important insights.

## ✨ Features

- Upload and analyze food quality datasets
- Interactive charts and graphs
- Statistical summary
- Data visualization
- Easy-to-use dashboard

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

## 📂 Dataset

The project uses a food quality dataset to generate visual insights and statistical summaries.
